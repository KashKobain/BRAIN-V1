# 🧠 BRAIN

**B**ehavioral **R**easoning **A**nd **I**ntelligent **N**etwork

23-skill cognitive architecture for Claude Code. Deployed 2026-04-12.

## What Is This?

BRAIN is a complete cognitive architecture that transforms Claude Code into a self-improving AI system with 23 explicit skills organized across 6 cognitive planes.

**This is not theoretical AI research.** This is operational infrastructure built under real conditions by someone who operates production systems daily.

## Architecture

### Six Cognitive Planes

The BRAIN architecture is structured as 6 functional layers that work together:

#### 1. Perception Plane
- **goal-extraction**: Extract core objectives from natural language
- **context-mapping**: Build comprehensive understanding of task environment
- **pattern-detection**: Identify relevant patterns in data/code/requirements

#### 2. Executive Plane
- **task-decomposition**: Break complex goals into executable subtasks
- **tool-sequencing**: Determine optimal order of tool usage
- **workflow-orchestration**: Coordinate multi-step processes
- **adaptive-planning**: Adjust plans based on intermediate results

#### 3. Control Plane
- **resource-management**: Optimize token usage and API calls
- **error-recovery**: Handle failures gracefully and retry intelligently
- **state-maintenance**: Track progress across conversation boundaries
- **constraint-enforcement**: Respect limits and boundaries

#### 4. Reasoning Plane
- **logical-inference**: Apply deductive and inductive reasoning
- **causal-reasoning**: Understand cause-effect relationships
- **hypothesis-generation**: Generate and test explanatory models
- **knowledge-synthesis**: Combine information from multiple sources

#### 5. Awareness Plane
- **metacognition**: Think about own thinking processes
- **progress-tracking**: Monitor advancement toward goals
- **capability-assessment**: Know what can and cannot be done
- **uncertainty-quantification**: Express confidence appropriately

#### 6. Judgment Plane
- **priority-evaluation**: Determine what matters most
- **quality-control**: Assess output correctness and completeness
- **tradeoff-analysis**: Balance competing objectives
